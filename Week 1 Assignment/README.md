# Shopping Trends Data Analysis

This project demonstrates basic data exploration and cleaning using Python and Pandas.

## Objective
Learn Python basics and perform fundamental data analysis tasks:
1. Load a CSV dataset into a Pandas DataFrame.
2. Explore data (head/tail, shape, columns, data types).
3. Handle missing values.
4. Perform basic operations (filter, select).
5. Remove duplicates.
6. Create derived columns.
7. Save the cleaned dataset.

## Setup Instructions

1. **Clone this repository** (or download the files).
2. **Download the Dataset:**
   - Go to Kaggle: [Shopping Trends Dataset](https://www.kaggle.com/datasets/anvitkumar/shopping-dataset)
   - Download the file and ensure it is named `shopping_trends.csv`.
   - Place `shopping_trends.csv` in the root folder of this repository.
3. **Run the Notebook:**
   - Install Jupyter and Pandas if you haven't already: `pip install jupyter pandas numpy`
   - Start Jupyter Notebook: `jupyter notebook`
   - Open `data_analysis.ipynb` and run the cells.

## Outputs
- `data_analysis.ipynb`: The main Jupyter Notebook.
- `cleaned_shopping_dataset.csv`: The cleaned dataset output (generated after running the notebook).
