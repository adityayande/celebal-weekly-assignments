# Week 2 Assignment: Sales Data Analysis using SQL

## Objective
Analyze sales data using SQL with filtering, aggregation, and business queries.

## Tasks Completed
1. Load dataset into an in-memory SQL database (SQLite).
2. Explore table (schema, sample data).
3. Apply WHERE filters (region, category, date, sales).
4. Use GROUP BY for aggregations (sales, quantity, averages).
5. Sort and limit results (top products, top categories).
6. Solve use cases (monthly trends, top customers, duplicates).
7. Validate results (row counts, data quality).

## How to Run
1. Ensure you have the Superstore dataset (`Sample - Superstore.csv`). You can download it from [Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final).
2. Place the CSV file in the `Week 2 Assignment` folder.
3. Run the Jupyter Notebook (`sql_sales_analysis.ipynb`). 
*(Note: If the CSV is missing, the notebook will automatically generate mock data to demonstrate the queries).*
